🔥 Step 1 — Firebase (Real-time Sync)

1. Go to console.firebase.google.com → Create a project (e.g. "NeonReel")

2. Add a Web App → copy the config object

3. Enable Firestore Database (test mode is fine to start)

🎬 Step 2 — TMDB API Key (Auto-fetch posters)

1. Go to themoviedb.org → Sign up (free)

2. Settings → API → Request an API key → choose Developer

3. Copy your API Read Access Token (the long one)

📱 APK for Android

To install this as an Android app:

1. Install Node.js from nodejs.org

2. Run: npm install -g @capacitor/cli

3. Create a project folder, copy tracker.html into www/index.html

4. Run: npx cap init NeonReel com.neonreel.app

5. Run: npx cap add android

6. Run: npx cap open android → Android Studio → Build APK

✅ Same HTML file works on both web and APK — same Firebase sync!
